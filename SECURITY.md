# Security Policy

## Supported Versions

Security updates are currently provided for:

| Version | Supported |
| --- | --- |
| `main` branch | Yes |
| Latest npm release (`@galihru/orbinexsim`) | Yes |
| Older releases | No |

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead, report responsibly through one of the following channels:

- GitHub Security Advisories (preferred)
- Email: g4lihru@students.unnes.ac.id

Please include:

- A clear description of the issue
- Reproduction details or proof of concept
- Impact assessment
- Suggested remediation (if available)

We will acknowledge your report as soon as possible and provide updates while
investigating.
